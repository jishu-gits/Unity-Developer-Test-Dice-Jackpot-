## 🎮 How to Play

### Option 1: Play the Standalone Build (.exe)
Due to GitHub's file size limits for builds, the playable Windows executable is securely hosted on Google Drive.

1. **[Download the Playable Build Here] (Insert-Your-Google-Drive-Link-Here)**
2. Extract the downloaded `DiceJackpot_Build.zip` folder.
3. Run the executable file (`.exe`) located inside the extracted folder.
4. Click the **Roll** button to toss the dice.

### Option 2: View the Source Code
1. Open Unity Hub and create a new 3D project (Unity 6000.x / Unity 6 recommended).
2. Download the `[YourName]_DiceJackpot.unitypackage` file directly from this repository.
3. Drag and drop the `.unitypackage` file into the Unity Editor's Project window and click **Import All**.
4. Open the primary scene to view the hierarchy, scripts, and component setup.
