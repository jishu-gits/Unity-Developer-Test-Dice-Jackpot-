## 🎮 How to Play

### Option 1: Play the Standalone Build (.exe)
Due to GitHub's file size limits for builds, the playable Windows executable is securely hosted on Google Drive.

1. **[Download the Playable Build Here] -  https://drive.google.com/drive/folders/1mVX6-4QpDLH38VDBALtYyUwYNQyqiPFW?usp=sharing **
2. Extract the downloaded `DiceJackpot_Build.zip` folder.
3. Run the executable file (`.exe`) located inside the extracted folder.
4. Click the **Roll** button to toss the dice.
5. .Unitypackage is also avalaible in the goggle drive folder
